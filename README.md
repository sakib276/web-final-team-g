# SpaceSync: A Resource Allocation System (Team G)

A full-stack web application built with a **React frontend** and an **Express backend**, organized into separate modules for better scalability and maintainability.

---

## 🚀 Project Structure

```bash
project/
│
├── backend/     # Express.js API
├── frontend/    # React application
├── package.json # Root scripts (combined commands)
└── README.md
```

---

## 🛠️ Tech Stack

### Frontend

* React
* JavaScript (ES6+)

### Backend

* Node.js
* Express.js
* Sequelize

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/sakib276/web-final-team-g.git
cd web-final-team-g
```

---

### 2. Install all dependencies (single command ✅)

From the root directory:

```bash
npm run install
```

This will install dependencies for:

* `backend/`
* `frontend/`


---

### 3. Run the project

#### Run both frontend & backend together:

```bash
npm run dev
```


---

### 4. Open in browser

Go to http://localhost:5173
